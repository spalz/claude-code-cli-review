# Review Tasks

## 1. Не открывать файлы автоматически при добавлении в ревью

Сейчас при появлении нового файла на ревью он автоматически открывается в редакторе. Нужно убрать это поведение — пользователь сам откроет файл когда будет нужно.

**Где искать**: `src/lib/review-manager/file-addition.ts` — `addFile()`, вероятно вызывает `openFileForReview` или навигацию после добавления.

## 2. Привязка изменённых файлов к сессии Claude Code

Связывать каждый файл попавший в ревью с конкретной сессией Claude (claudeId). Цель — показывать список изменений сгруппированный по сессиям.

**Что нужно**:
- Hook уже передаёт `session_id` в JSON payload (`/changed`, `/snapshot`)
- Сохранять `sessionId` в `IFileReview` или отдельной Map
- UI: группировка файлов по сессиям в списке ревью
- Возможность просмотреть все изменения конкретной сессии
